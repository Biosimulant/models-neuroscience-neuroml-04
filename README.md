# models-neuroscience-neuroml-04

Curated collection of **neuroscience NeuroML** models (Part 4 of 4) for the **biosim** platform. This repository contains 404 NeuroML models of neurons, ion channels, synapses, and neural circuits from the NeuroML model database.
